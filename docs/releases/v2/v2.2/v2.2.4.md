## Description of Changes

<!-- user-editable-section-start -->
- Add the `PYTHONPATH` environment variable to the `internal media generate` command
  - Without this, media generation would fail once helper modules were introduced.
  - This happens because the Manim CLI loads scene files individually using a custom module loader, which means the full project context is not automatically included in Python's module resolution.
  - By setting `PYTHONPATH` to the `src` directory, imports such as helper utilities can be resolved correctly during rendering.
