## Description of Changes

<!-- user-editable-section-start -->
- Lazy import the birthday data in the birthday notifications.
  - Otherwise people on older versions of Node than Node v26 will get an error, as the import would be resolved at module load time rather than when the function itself runs.
  - This changes that so that we instead dynamically import inside the module itself.
