## Description of Changes

<!-- user-editable-section-start -->
- Add license reporting to `internal dependency-audit`
  - This will report back on the current state of the licenses of our dependencies, and highlight any licenses that may require review.
  - It won't fail/block us if we have any licenses requiring review, but we will at least be made aware.
  - Eventually I would like to make the review criteria configurable, which is something I plan to do once I feel we're in a good position to promote the command to being a main command rather than internal.
